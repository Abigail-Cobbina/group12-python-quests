##  About This Repository

This repository contains Group 12's work for the Python Basics Peer-Reviewed Activity. Each quest is a separate Python script that focuses on a specific programming concept such as variables, user input, conditionals, loops, and functions.

The quests are organized into six levels, with each level becoming more challenging than the previous one. Through these activities, we learned how to write Python programs, solve problems using code, and work together as a team using GitHub.

This project helped us strengthen our programming skills while gaining experience with version control, collaboration, and code review.

##  Running the Programs

Make sure Python 3 is installed:

```bash
python3 --version
```

Run any quest using:

```bash
python3 quests/quest_18_loop_of_riddles.py
```

Some quests are interactive and will ask for user input in the terminal.
